## Overview

AsaanLang is a localized programming language designed to lower entry barriers for South Asian developers. It features a custom lexer and parser that transpiles Urdu/Hindi-based keywords directly into Python, leveraging its robust ecosystem and dynamic typing.

## Key Features

- **Localized Syntax**: Uses Latin-based Urdu/Hindi keywords for logic and flow control.
- **Web-Based IDE**: Integrated code editor built for writing and testing scripts in-browser.
- **CLI Support**: A dedicated command-line interface for local compilation and execution.

## Technologies

- **Backend**: Python, Flask (Transpilation Engine)
- **Frontend**: React.js, Vite, Tailwind CSS
- **Tools**: Git, GitHub

## Impact

Successfully created a working proof-of-concept that maps native language logic to Python's AST, proving that localized syntax can significantly reduce the initial learning curve for non-English speaking students.
