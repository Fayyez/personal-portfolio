## Overview

Countistics is a computer vision solution designed for real-time traffic monitoring. It utilizes a fine-tuned object detection model to track and count vehicles, providing actionable analytics for parking and traffic flow management.

## Key Features

- **YOLOv8 Optimization**: Fine-tuned on a custom dataset of Pakistani vehicle types for high localized accuracy.
- **Real-Time Processing**: Leverages OpenCV for low-latency video analysis and vehicle tracking.
- **Interactive Dashboard**: Streamlit frontend for live data visualization and historical reporting.

## Technologies

- **AI/ML**: YOLOv8, OpenCV, Python
- **Frontend**: Streamlit
- **Tools**: Git, GitHub

## Impact

The project won recognition for its practical utility and was deployed at COMSATS University to automate campus traffic counting and optimize parking occupancy.
