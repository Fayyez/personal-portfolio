## Overview

Hazir-io automates the attendance process using facial recognition. It provides a seamless interface for educators to manage students and generate automated attendance logs in real-time.

## Key Features

- **Face Recognition API**: Uses high-level facial encoding for fast and accurate student identification.
- **Desktop Management**: Professional GUI built with PyQt5 for managing class rosters and viewing logs.
- **UX Prototyping**: Designed with Figma and Balsamiq to ensure an intuitive teacher-centric workflow.

## Technologies

- **AI/CV**: OpenCV, Python, Face-recognition
- **GUI Framework**: PyQt5, PyQt5 Designer
- **Design**: Figma, Balsamiq

## Impact

Reduced manual roll-call time by 90% in testing environments, showcasing a production-ready application of computer vision in education.
