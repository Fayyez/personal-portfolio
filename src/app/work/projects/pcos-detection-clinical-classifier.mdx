## Overview

This project addresses the need for non-invasive diagnostic tools in biomedicine. It uses various machine learning algorithms to classify PCOS risk based on patient clinical history, offering a fast and low-cost alternative to imaging.

## Key Features

- **High-Precision Classification**: Utilizes Scikit-learn to compare various ML models for the best sensitivity/recall balance.
- **Exploratory Data Analysis**: Extensive use of Seaborn to visualize correlations between clinical biomarkers and PCOS.
- **Biomedical Impact**: Optimized for early screening to assist medical professionals in clinical decision-making.

## Technologies

- **Languages**: Python
- **Libraries**: Scikit-learn, NumPy, Pandas, Seaborn
- **Environment**: Google Colab

## Impact

Won the Soliton Technology AI in Biomedicine bootcamp competition for its high accuracy and potential to democratize healthcare screenings.
