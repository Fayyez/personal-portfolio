## Overview

PlagPatrol is a web-based utility designed for academic integrity. It uniquely handles both digital text and handwritten assignments by integrating OCR technology with a web-scraping comparison engine.

## Key Features

- **Handwriting Support**: Integrated OCR engine to extract and verify text from physical document photos.
- **Automated Reporting**: Generates detailed PDF plagiarism reports including source links and similarity scores.
- **Web Scraping**: Custom engine to verify content against online academic sources.

## Technologies

- **Backend**: Python, Flask
- **PDF Generation**: FPDF
- **Frontend**: HTML5, CSS3, JavaScript
- **OCR**: Python-based OCR libraries

## Impact

Delivered a high-utility academic tool that bridges the gap between traditional paper-based submissions and digital plagiarism detection.
