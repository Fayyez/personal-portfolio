## Overview

WeatherWatch is a Java-based desktop application that fetches real-time data via the Open Weather API. The project serves as a showcase for architectural excellence, prioritizing modularity and the SOLID principles.

## Key Features

- **Strategy Pattern Architecture**: Supports hot-swappable UI (CLI/GUI) and database layers (File-based/MySQL).
- **Desktop GUI**: Built with JavaFX and Scene Builder for a modern, responsive user experience.
- **Dependency Management**: Fully managed via Maven for scalable builds.

## Technologies

- **Language**: Java (JDK 17+)
- **UI/UX**: JavaFX, Scene Builder
- **Storage**: MySQL, File I/O
- **Tools**: Maven, IntelliJ IDEA

## Impact

Demonstrated the ability to build highly decoupled systems where the business logic remains independent of the delivery mechanism (UI) or persistence layer (Database).
